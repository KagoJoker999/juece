# 方案利润计算器

一个简洁的网页工具，帮助快速计算不同处理方案的利润损失，辅助决策制定。

## 在线访问

🌐 **[立即使用](https://kagojoke999.github.io/juece/)**

## 功能特点

- 📱 **移动端友好** - 完全响应式设计，支持手机纵向浏览
- 🧮 **实时计算** - 输入售价后自动计算各方案损失
- 📊 **智能排序** - 方案按损失从小到大自动排列
- 💾 **本地保存** - 执行方法可编辑并自动保存到本地
- 🎨 **简洁设计** - 采用Google Material Design风格

## 使用方法

1. 在"售价输入"区域输入商品售价
2. 系统自动计算四种处理方案的损失金额
3. 点击选择最优方案查看执行方法
4. 可编辑并保存执行方法和注意事项

## 方案计算公式

- **退货退款**：损失 = 0 - (售价/2) - 7
- **补发**：损失 = 0
- **仅退款**：损失 = 0 - (售价/2)
- **换货**：损失 = 售价 - (售价/2 + 10)
